KADER ARSLAN

Necmettin Erbakan Üniversitesi Bilgisayar Mühendisliği Bölümünden mezunum. Bölüm dersimiz olan Görsel Programlama dersinde C# ve SQL Server veri tabanını kullanarak Hastane Otomasyonu yaptığım sıralarda arayüz tasarımına ilgili olduğumu farkettim. Daha sonra Bitirme projemde Python ve Kivy ile Mobil Arayüz tasarımında ilgilimin doğru yöne ilerlediğini ama web ve mobil arasında hangisine yönelmem gerektiğine karar vermem gerekti. Araştırmalar sonucu JavaScript teknolojileri ile Web Frontend Developer olma yoluna girdim. JavaScript teknolojilerinden React ile başladım. Hedefim Doğrultusunda Kodluyoruz'un Full Stack Web Developmment Bootcamp'inde kendimi geliştirebileceğimi düşündüm. Bootcamp sürecinde teknik bilgilerden ziyade pratik yapma fırsatı yakaladım. En önemliside ekip olarak agile ve scrum yazılım geliştirme metodojileri ile çalışarak proje geliştirdik. Bootcamp sürecinde Full Stack konuları dahilinde Nodejs ve Mongodb ile de ilgilendim. Kendimi Frotend Developer alanında iyi seviyeye geldikten sonra Full Stack Web Developer olarak kariyerime devam etmeyi düşünüyorum. Aşağıda teknolojiler ve yaptığım projeler yer almakta.
